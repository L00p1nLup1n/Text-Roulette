# Text-Roulette

A 2-player, text-based version of Buckshot Roulette.

## Requirements
- Java 8 or higher
- Maven 3.6+

## Build & Run
```bash
# Clone repository
git clone https://github.com/L00p1nLup1n/Text-Roulette.git
cd Text-Roulette

# Build fat JAR
mvn clean package

# Run the game
java -jar target/text-roulette-1.0.0.jar
